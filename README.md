# solidity-starter
Starter project for solidity smart contract

## Version

**Node.js**: ^12.22.3

**npm**: ^6.14.13

## Install Dependencies

```
npm i
```

## Deploy contracts
Rinkeby - Ethereum testnet
```
npm run deploy:rinkeby
```

BSC testnet
```
npm run deploy:testnet
```

Ethereum mainnet
```
npm run deploy:eth
```

BSC mainnet
```
npm run deploy:bsc
```

## Test

```
npm run test
```
